Construction Website

In a team of 2, my partner and I created a website for a local construction company using the MVC pattern and the ASP.NET framework. This website was built using Visual Studio, HTML, CSS, JavaScript, and C#.

The client ended up not using the website, but a temporary url of it can be found here: http://apostle2022-001-site1.itempurl.com/

This website was our Capstone Project for college. As such, we built it free-of-charge for the company. The above image is the About Section for the company's website. Some of the services they provide include entirely new construction, exterior and interior painting, tape and float services, as well as renovations and repairs.

A Contact Section was included in the Home Page. This section provided users with the company's phone number, email address, and Facebook page so that they could easily contact the company if they needed construction work done.

This webpage showcases some of the projects completed by the company. Clicking on one of these projects sends the user to a page solely dedicated to describing that project. An MS SQL database was used to store data pertaining to each project. This data was then used to populate the above web page and all other pages that showcase projects.

Here is the first half of a project's page. It provides the name of the project, its address, and a picture of the finished product.

Here is the second half of the project's page. It provides the type of construction performed, the scope of the project, and information about the cost of the project as well as it's completion date.

This is an Employee Portal page. It uses an API provided by OnTheClock that lets employees of the company clock-in and clock-out of work. This page also offers helpful documents for current employees as well as aspiring employees.

For more information about OnTheClock, visit their website: https://www.ontheclock.com/time-clock/

We also created an Admin Console for the website! This is the Admin Login page.

Once logged-in, admins can create, edit, or delete construction projects that are visible on the website. The above image shows the first half of a form used by admins to add a new project to the website.

This is the second half of the form that allows admins to add new projects. Once the admin clicks the 'create' button, the new project will be stored in the MS SQL database and will be visible on the company's website.

Shopping Website

As a project for college, my partner and I created a website for a fake wholesale grocery store that was meant to mimic Sam's Club's membership-based model, in which customers must be members in order to shop at the store.

This project's GitHub repository can be found here: https://github.com/charlesQuigley/GroceryStore

This project was built in Visual Studio and used the ASP.NET framework. This website consists of HTML, CSS, JavaScript, and C#.

In order to begin shopping, users must first log into their membership accounts.

Once they have logged-in, users will be redirected to the Home Page. The icon on the top-right of the page will turn green, indicating that the user is now signed-in and can begin shopping.

The user can now click on any of the products shown on the Home Page. Once the user clicks on a product, the user will be redirected to that product's page.

For example, if the user clicks on "Paper Towels", the user will be redirected to a page that provides a description of the product as well as it's price.

The user can now choose to add the product to their online shopping cart.

Users can also find products using the search bar at the top of the page. For example, entering "peanut" in the search bar will generate a page that displays all items with the word "peanut" in their name.

This page is populated by a database. Each database row is searched for the word "peanut". If the word is found, that row is sent to the search page.

Users can click on the "Cart" button at the top of the page to view the items currently in their shopping carts. This page shows which products the user is about to purchase, how many of each product will be purchased, and the total price of their purchase.

Click the above image to view the entire Shopping Cart page.

If the user decides to buy all the items in the shopping cart, they will then be redirected to a page that shows their online receipt. They will then pick-up their items from the store's physical location on the date specified by the receipt.
